插 件 简 介

fixedNav.js 是一款超轻量级的 javascript 插件(无依赖)。


它可以让原本不是在顶部的导航条，


在滚动到它的位置时固定在顶部。


就像本页的导航条一样 ^_^

下 载 方 法

  • 下载 fixedNav.js

    这是未经压缩的版本，您可以愉快地阅读它的代码。欢迎与作者交流学习。

  • 下载 fixedNav.min.js

    这是经过压缩的版本，您可以在实际的生产环境中使用它，以节省数据流量。

  • Visit GitHub

    navFixed 是一款开源的插件，您可以到 GitHub 上浏览它。

安 装 指 南

1.引入 fixedNav.js 。
2. 然后在 window.onload = function(){} 里使用 new fixed({elem:'#nav'}) #nav你的选择器 即可。

<script src="fixedNav.min.js"></script>
<script>
    window.onload = function(){}
        new fixed({
        	elem:'#nav'
        })
    };					
</script>
